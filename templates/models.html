Our Machine Learning Models

Convolutional Neural Network (CNN)

Our CNN model is specifically designed for image classification tasks. It consists of multiple convolutional layers followed by max-pooling layers, and finally dense layers for classification.

Accuracy: 94.2% Precision: 93.8% Recall: 94.1%

K-Nearest Neighbors (KNN)

The KNN model classifies images based on the similarity to other images in the feature space. We use k=5 neighbors with Euclidean distance metric.

Accuracy: 88.5% Precision: 87.9% Recall: 88.3%

Support Vector Machine (SVM)

Our SVM model uses a radial basis function (RBF) kernel to classify the images in a high-dimensional feature space.

Accuracy: 90.1% Precision: 89.7% Recall: 90.0%

Random Forest

The Random Forest model uses an ensemble of decision trees to improve classification performance and reduce overfitting.

Accuracy: 91.3% Precision: 91.0% Recall: 91.2%

Ensemble Model

Our ensemble model combines predictions from all the above models using majority voting to achieve the highest possible accuracy.

Accuracy: 95.7% Precision: 95.5% Recall: 95.6%
